Pitcher Post-Outing Report

Immediate post-game analysis with historical comparison

Project Overview

Using R as a tool and Trackman CSVs as my data, I was able to craft a report that analyzes a pitcher's outing in comprehensive detail, providing immediate feedback for coaching staff and players.

R Programming Trackman Data Post-Game Analysis Historical Comparison

Project Description

This report was created with the goal of evaluating a pitcher's outing immediately following their performance. It provides a complete picture of what happened during the game, combining traditional statistics with advanced metrics and detailed matchup analysis.

Outing Summary Metrics

The report begins with a comprehensive summary table featuring key performance indicators including pitches thrown, batters faced, strikeouts, walks, hits, extra-base hits, max velocity, whiffs, strike percentage, CSW%, pitch win performance in 3-2 and 1-1 counts, and negative to positive transitions.

Report Structure

1

Performance Summary

Traditional and advanced statistics including pitch counts, results, strike percentage, CSW%, and count-specific performance metrics.

2

Pitch Metrics Table

Detailed analysis of each pitch type including usage, velocity, spin, extension, break, tilt, zone-adjusted VAA, release point, and arm angle approximation.

3

Movement & Release Plots

Visual representations of pitch movement patterns with arm angles and release point consistency across the arsenal.

4

Results Analysis

Zone percentage, chase rate, and batted ball results (ground balls, line drives, fly balls) organized by pitch type.

5

Historical Comparison

Side-by-side comparison of this outing versus the pitcher's last 5 appearances for pitch metrics, release attributes, and results.

6

Location Analysis

Comprehensive pitch location visualizations showing both general locations and specific whiff locations by pitch type.

7

Matchup Breakdowns

Individual pages for each batter faced, with pitch-by-pitch analysis including results, locations, and metrics for every plate appearance.

One of the most valuable aspects of this report is the comparison table that shows important pitch metrics, release attributes, and results statistics alongside the differences between this outing and the pitcher's last 5 appearances. This is particularly helpful for noticing changes that could predict a dip or jump in results, or potentially identify early warning signs of injury. By tracking these trends, coaches can make informed decisions about pitcher health and performance adjustments.
