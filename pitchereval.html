Pitcher Evaluation Report

Comprehensive stuff analysis using Trackman metrics

Project Overview

Using R as a tool and Trackman CSVs as my data, I was able to craft a report that analyzes pitchers and evaluates their stuff through detailed pitch metrics, movement patterns, and location analysis.

R Programming Trackman Data Pitch Movement Analysis

Project Description

This report was created with the goal of evaluating a pitcher and his stuff. The report provides a comprehensive view of each pitcher's arsenal, combining statistical tables with visual representations to give coaches and analysts actionable insights into pitch quality, movement profiles, and command patterns.

Pitch Metrics Analyzed

Usage
Pitch type frequency and distribution across different batter handedness
Velocity
Average and range of pitch speeds for each pitch type
Spin Rate
Rotational speed of pitches
Extension
Release point distance from rubber, affecting perceived velocity
Break
Horizontal and vertical movement patterns for each pitch
Tilt
Clock-based orientation of spin axis, indicating pitch shape
VAA (Zone-Adjusted)
Vertical approach angle adjusted for different zone locations by pitch type
Release Point
Horizontal and vertical coordinates of ball release
Arm Angle Approximation
Calculated arm slot based on pitcher height and release point
Results
Outcomes including swings, whiffs, and chases

Visual Components

Pitch Movement Plot

Comprehensive movement chart showing horizontal and vertical break for each pitch type, with arm angle annotations to contextualize release mechanics.

Release Point Plot

Visual representation of release consistency across pitch types, helping identify deception opportunities and mechanical consistency.

General Location Plots

Heat maps and scatter plots showing where pitches are being located, split by pitch type to reveal command patterns and approach strategies.

Whiff-Specific Plots

Detailed analysis of pitch locations that generate swings and misses, segmented by pitch type to identify optimal attack zones for each offering.

Adaptable Analysis

The attached report includes all of the data from the 2025 Iowa baseball season. However, the code is designed to work with any Trackman data, making it a versatile tool for analyzing pitchers across different teams, leagues, and competitive levels.
